    DESCRIPCION

    discusion de conceptos de diseno y desarrollo de paginas de web. aplicacion de los lenguages HTML y JavaScript conjuntamente con CSS para la creacion de paginas web. creacion y adaptacion de elementos de multimedio para ser utilizado en paginas interactivas. requiere 45 horas de conferencia-laboratorio. 3 creditos

    OBJETIVOS

    se espera que finalizar el curso, el estudiante pueda:

  1. discutir conceptos de diseno y desarrollo de paginas de web.
  2. aplicar los lenguages HTML y JavaScript conjuntamente con CSS para la creacion de paginas de web.
  3. crear y adaptar elementos multimedios para ser utilizadas en paginas de interactivas.
  4. entender las implicaciones eticas,sociales y legales inherentes a la disciplina.
